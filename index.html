Huafu AS3 Library

This library is full of many AS3 classes that can be used to manipulate data/visual elements in a relatively quick and easy way. But the main big stuff is the ORM and ActiveRecord like addition that it provides with help of the com.huafu.sql.orm package.

The API doc is on GitHub pages here.

It's still in early (even if quite big already) development stage so more help coming up after the first beta version will be tagged.
